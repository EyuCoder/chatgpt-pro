# ChatGPT clone

> a demo of the App

  <img src="screenshots/demo.gif" width="800px" alt="android icon"/>


## install

```bash
npm install
```

## Configuration
1. obtain you openai api key from [here](https://openai.com)
2. `cd server`
3. rename `.env.example` to `.env`
4. add your openai api key inside `.env`

## run
```bash
npm run dev
```


***Tech used***
  - openai
  - react with vite
  - tailwindcss
  - react-icons
  - react-markdown
  - express
