## install

```bash
npm install
```

## Configuration
1. obtain your openai api key from [here](https://openai.com)
2. `cd server`
3. rename `.env.example` to `.env`
4. add your openai api key inside `.env`
5. make sure you have added `.env` to your `.gitignore` file

## run
```bash
npm run dev
```
